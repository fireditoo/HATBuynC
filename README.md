# 项目简介

## 核心功能
- 用户端：
  - 首页浏览
  - 个人中心管理
  - 订单评价查看
  - 我的收藏管理
  - 购物车操作
  - 资讯信息浏览
  - 订单管理
- 管理端：
  - 用户管理
  - 零食分类管理
  - 零食信息管理
  - 订单评价管理
  - 系统管理
  - 订单管理

## 适用场景
- 零食电商平台
- 企业内部管理系统
- 食品零售行业信息化建设

## 技术特点
- 使用Springboot框架构建后端，提高开发效率和稳定性
- 前端采用Vue.js和Element UI，提供丰富的UI组件和便捷的开发体验
- 使用MyBatis进行数据库操作，简化数据库访问
- 支持JDK1.8和MySQL8，确保项目兼容性和稳定性
